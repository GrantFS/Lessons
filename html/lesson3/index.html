Responsive

Responsive is a term used to describe the ability to scale depending on the screen size.

The main devices are

  1. Older Mobile Devices (col-)
  2. Mobile (col-sm)
  3. Tablet (col-md)
  4. Laptop (col-lg)
  5. Retina screen - HD/UHD/4K/iMac (col-xl)

Bootstrap Responsive columns
  • The class col will automatically use the available space.
  • The prefix col- will work below a screen that is below 576px wide.
  • The prefix col-sm will work from a screen that is 576px wide.
  • The prefix col-md will work from a screen that is 768px wide.
  • The prefix col-lg will work from a screen that is 992px wide.
  • The prefix col-xl will work from a screen that is 1200px wide.
  • The -auto will automatically use the remaining space.
