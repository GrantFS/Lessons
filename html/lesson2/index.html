HTML

HTML Tags

  • HTML tags are element names surrounded by angle brackets.
  • HTML tags normally come in pairs
  • The first tag in a pair is the start tag
  • The second tag is the end tag
  • The end tag is written like the start tag, but with a forward slash inserted before the tag name

HTML elements

An HTML element usually consists of a start tag and an end tag, with the content inserted in between.

HEAD Element

  • This contains metadata
  • This is specific information about the web page that is not displayed to the user.

This can contain:

  1. Document
  2. Title
  3. Character set
  4. Styles
  5. Links
  6. Scripts

BODY Element

This defines the document's body.

This contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc

FOOTER Element

This is a structural element used to identify the footer of a page.

This typically contains copyright and authorship information.

It can also contain navigational elements pertaining to the contents of the parent element

  1. Document
  2. Article
  3. Section

Other information

The DOCTYPE

  • It must be the very first thing in your HTML document, before the html tag.
  • Is not an HTML tag
  • it is an instruction to the web browser about what version of HTML the page is written in.

Metadata

Metadata provides browsers and search engines with technical information about the web page.

HTML Entities

Some characters are reserved in HTML.

If you use the less than (<) or greater than (>) signs in your text, the browser might mix them with tags.

  • < - Less than
  • > - Greater Than
  • & - Ampersand
  • 1 2 - Non Breaking Space
  • " - Double quotation
  • ' - Single quotation
  • £ - Pound
  • © - Copyright
  • ® - Registered trademark
